## HEART WEB - BACKEND

### Development

Additional dependencies:

- [Python >=3.8](https://www.python.org/downloads)

- [Poetry](https://python-poetry.org)

### Installing

To install the dependencies:

`poetry install`

### Running

- Start the app: `uvicorn main:app --reload`

### Testing

- Run test: `pytest`

By default the server starts on port 5000.
